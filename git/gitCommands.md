
## Inicializacija projekta
Inicializacija repozitorija v mapi
```bash
init
```

## Commit
Dodaj datoteke za commit
```bash
git add .
```

Commit sprememb
```bash
git commit -m "opis"
```

## Branches/veje
Izpis vej:
```bash
git branch
```
Zamenjava veje (-b usvari novo vejo):
```bash
git checkout -b imeVeje
```
